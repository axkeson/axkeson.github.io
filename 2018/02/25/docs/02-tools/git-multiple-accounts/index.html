Git 客户端多账号管理

发表于 2018-02-25 分类于 工具

前言

在开发的过程中，经常会遇到这样的状况：需要在一台电脑上同时使用两个甚至多个 git 账号，负责不同的用途，比如：一个用来写个人项目，一个用来写公司的项目。为此我们需要为不同的账号生成不同的密钥，那对这些不同的账号和不同的密钥，我们该怎么处理呢？

在这里我创建了两个目录 github 和 company ，分别用来存储不同项目的密钥，进行分类管理

New SSH key

  1. 把 ~/.ssh/github/id_rsa_github.pub 的内容添加到Github的SSH keys中
  2. 把 ~/.ssh/company/id_rsa_company.pub 的内容添加到公司Gitlab的SSH keys中
